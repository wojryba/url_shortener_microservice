URL Shortener Microservice

User stories:

  1. I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  2. When I visit that shortened URL, it will redirect me to my original link.

Example creation usage:

https://whispering-harbor-61728.herokuapp.com/new/https://www.google.com
https://whispering-harbor-61728.herokuapp.com/new/https://www.wp.pl

Example creation output:

{"original":"https://www.google.com","short":"localhost/58c6fa8a0fd3d436c4336056"}}

Usage:

https://whispering-harbor-61728.herokuapp.com/58c6fa8a0fd3d436c4336056

Will redirect to:

https://www.google.com/
